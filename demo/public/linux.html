Linux 常用命令

文件目录

ls -alh 详细列表，人类可读大小
cd - 返回上一次目录
pwd 显示当前路径
mkdir -p a/b/c 递归建目录
rm -rf dir 强制删目录
cp -r src dst 递归复制
mv old new 移动/重命名
touch file 新建空文件或更新时间戳

文件查看与编辑

cat file 全量输出
less file 分页查看，可搜索
head -n 20 file 看前 20 行
tail -f log 实时追踪日志
grep -i 'err' log -C3 忽略大小找并显示前后 3 行
awk '{print $1}' access.log | sort | uniq -c | sort -nr | head 统计 IP 前 10
sed -i 's/old/new/g' *.conf 批量替换

进程与资源

ps -ef | grep java 查 Java 进程
top -H -p PID 查看某进程所有线程 CPU
htop 彩色 top，需安装
kill PID 优雅终止
kill -9 PID 强制杀
jstack PID > jstack.log 导出线程栈
jmap -dump:format=b,file=heap.hprof PID 导出堆
ulimit -n 查看文件句柄上限

网络

netstat -tunlp | grep 8080 端口占用
ss -lntp 更快替代 netstat
curl -I URL 查看响应头
curl -X POST -d 'k=v' URL 发 POST
wget URL 下载文件
telnet ip port 测端口连通
nc -zv ip 1-65535 批量扫端口

磁盘与系统

df -h 查看挂载与使用率
du -sh * 统计当前目录大小
free -m 内存占用
uptime 系统负载
whoami / id 当前用户/权限

权限与压缩

chmod +x script.sh 加执行权
chown -R user:group dir 改属主
tar czvf a.tar.gz dir 打包并压缩
tar xzvf a.tar.gz 解压
zip -r a.zip dir && unzip a.zip zip压缩与解压

系统服务（CentOS7+/Ubuntu16+）

systemctl start nginx 启动服务
systemctl enable nginx 开机自启
systemctl status nginx 查状态
journalctl -u nginx -f 实时看服务日志

SSH & 终端

ssh user@host 远程登录
scp file user@host:/path 上传文件
rsync -avz local/ remote:/dir/ 增量同步
tmux new -s work 新建会话
tmux a -t work 恢复会话

环境变量

export PATH=$PATH:/opt/bin 临时生效
echo 'export PATH=$PATH:/opt/bin' >> ~/.bashrc 永久生效
source ~/.bashrc 立即加载

实用快捷键

Ctrl + a / e 行首/行尾
Ctrl + r 反向搜索历史
!! 重复上一条命令
!$ 用上条最后一个参数

20 条以内即可覆盖日常开发 90% 场景，忘记参数时 man cmd 或 cmd --help。
